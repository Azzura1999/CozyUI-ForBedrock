# CozyUI+ For Bedrock

> 将 CozyUI+ 移植至《我的世界》基岩版的 UI 资源包。

---

## 📖 简介

**CozyUI+ For Bedrock** 是由将 Java版 CozyUI+ 移植到《我的世界》基岩版的 UI 资源包，致力于让基岩版玩家也能享受到 CozyUI 系列简洁、舒适的界面风格。

## 📦 安装方法

1. 下载 `.mcpack` 文件
2. 点击文件，等待游戏自动导入
3. 进入游戏 → 设置 → 全局资源 → 激活资源包
4. 重启游戏即可生效
